# Descripción del Proyecto

Este directorio contiene el código fuente original del Dashboard Estratégico desarrollado como Trabajo Final de la asignatura. El proyecto es una aplicación web interactiva de una sola página (SPA) construida con la biblioteca React y JavaScript, diseñada para el análisis operativo del dataset "DataCo Smart Supply Chain".

# Requisitos Previos del Sistema

Para ejecutar este proyecto de código fuente de manera local, es indispensable contar con el siguiente entorno de desarrollo instalado en su equipo:

Node.js (Versión 16.0 o superior)

NPM (Node Package Manager, incluido con Node.js)

# Instrucciones de Instalación y Ejecución

Paso 1: Restaurar dependencias
Al ser un proyecto de React, la carpeta de módulos (node_modules) ha sido omitida en el repositorio para optimizar el almacenamiento. Para descargar las dependencias necesarias, abra su terminal o consola de comandos, navegue hasta la raíz de esta carpeta y ejecute el siguiente comando:

npm install

Paso 2: Ejecutar el servidor de desarrollo
Una vez que la instalación de dependencias finalice, ejecute el siguiente comando para iniciar el dashboard en su entorno local:

npm start
(Nota: Si el proyecto fue inicializado con Vite, el comando será npm run dev)

Paso 3: Visualización
El servidor de desarrollo abrirá automáticamente su navegador web predeterminado. En caso de no hacerlo, ingrese manualmente a la siguiente dirección:

http://localhost:3000
